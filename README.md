# KeenKeeper

A friendship tracking app to stay in touch with people I care about.

Live site: https://mellow-daffodil-fe2ee4.netlify.app/

## Tech used
- React (Vite)
- React Router DOM
- Tailwind CSS
- Recharts
- React Hot Toast

## Features
- Friend dashboard with contact status
- Log check-ins (Call, Text, Video) from each friend's page
- Timeline of all past interactions
- Pie chart showing interaction breakdown

## Run locally
npm install
npm run dev
